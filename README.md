---

## ⚙️ Configurações e Práticas Aplicadas

Ao longo dos capítulos, foram aplicadas configurações e práticas comuns em ambientes reais de desenvolvimento backend:

- 🔹 Configuração por perfis (dev / test)
- 🔹 Uso de banco em memória (H2) para testes
- 🔹 Integração com PostgreSQL em ambiente de desenvolvimento
- 🔹 Versionamento de banco com Flyway
- 🔹 Logging estruturado com rotação de arquivos (Logback)
- 🔹 Ajuste de níveis de log para observabilidade (SQL, HTTP, parâmetros)
- 🔹 Documentação de API com SpringDoc (OpenAPI / Swagger)
- 🔹 Externalização de configurações via variáveis de ambiente
- 🔹 Customizações de `application.properties`
- 🔹 Configuração de locale e encoding

Esses pontos foram explorados progressivamente durante a evolução dos capítulos, com o objetivo de aproximar o ambiente de desenvolvimento de cenários reais de aplicação.
